# EduTech Egypt - Project Guide (CLAUDE.md)

## Tech Stack

- **Framework**: Next.js (App Router)
- **Backend / Database**: Supabase
- **Styling**: Tailwind CSS

## Project Rules

- **Curriculum**: All content must strictly support the **Egyptian Ministry of Education** curriculum (K-12).
- **Internationalization**: Use **RTL (Right-to-Left)** styling and logic for all Arabic text. Ensure layout consistency when switching between LTR and RTL.
- **Role Scoping**: Maintain clear boundaries between Student and Teacher flows.

## Collaboration

- **Antigravity**: Handles high-level missions, system architecture, database migrations, and core integrations.
- **Claude Code**: Handles complex business logic and high-fidelity UI components.
